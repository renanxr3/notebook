
## Util

| Method | Description |
|--------|-------------|
| Math.random() | Gera número randomico entre 0 e 1. parseInt(Math.random() * 10) = gera número base 10 |
| Math.round() | Arredonda para o número inteiro mais próximo |
| Math.ceil() | Arredonda para cima |
| Number.isInteger() | Verifica se é um número inteiro |
| numero.toFixed(2) | Retorna string com número fixado a 2 casas decimais |
| numero.toString(2) | Converte número para string em base 2 (binário) |
| Date.now() | Retorna timestamp atual em milissegundos |
| Date.getFullYear() ou Date.getUTCFullYear() | Retorna o ano da data |
| Date.parse() | Converte string de data para timestamp |
