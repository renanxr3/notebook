## ORM

Bancos de Dados & ORMs
Lista de tecnologias para persistência de dados (Backend e Mobile):

- TypeORM.io / Type ORM
- CouchDB
- Sequelize
- PouchDB
- Realm.io
- Knex JS (Query Builder para SQL)

> Conexão com seus projetos:
> Knex JS / TypeORM: São ótimas escolhas para o seu backend em Node.js, especialmente se você estiver trabalhando com bancos relacionais para o seu projeto de app de apostas.
