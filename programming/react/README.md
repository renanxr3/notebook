## Bibliotecas React

- Toastify: Para notificações flutuantes ("Toasts").
  - No App.js:
    `<ToastContainer autoClose={3000} hideProgressBar />`
  - No código:
    `toast.success("msg")`
- React i18next: Focada em tradução (internacionalização) de aplicações.

> Conexão com seus projetos:

- React i18next: Se o seu site pessoal v2 for voltado para o mercado internacional ou se você quiser exibir seu portfólio em inglês e português, esta é a biblioteca padrão para isso.
- Toastify: É uma das formas mais simples de dar feedback visual ao usuário quando uma ação (como "Salvar" ou "Erro") acontece no frontend.
